# Archived

`index.html` is the original **static, no-backend** version of the 2028 tier list.
Sharing works via a copy/paste base64 code — no server required, runs anywhere
(including GitHub Pages).

The active app lives at the repo root: `2028.html` (served by `server.py`),
which adds server-backed share links, a theme toggle, an image share generator,
and a read-only shared view.
